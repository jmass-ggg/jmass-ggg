# 💫 About Me:
Hi, I’m James Gurung, a backend developer who enjoys building the “invisible engine” behind real products — APIs, databases, authentication, deployment pipelines, and all the backend logic that makes an app actually work.<br><br>I have hands-on experience building and testing RESTful APIs using FastAPI and Django REST Framework, with strong skills in Python, PostgreSQL, JWT authentication, Docker, Pytest, Postman, Selenium, and Locust. I’ve worked on real-world projects including a hotel booking platform, a care marketplace web/mobile app, an LMS, and a multi-vendor e-commerce system.<br><br>I like backend development because it combines problem-solving, system design, debugging, and performance thinking. Whether I’m designing database models, securing APIs with role-based access control, writing tests, or deploying with Nginx, Gunicorn, Docker, and DigitalOcean, I enjoy making systems clean, reliable, and production-ready.<br><br>Outside of coding, I’m currently pursuing a Bachelor of Computer Applications at Southwestern State College, where I ranked 1st in both semesters. I also enjoy sharing knowledge and have conducted a WordPress training session for college students.<br><br>I’m always learning, building, and improving — one API, one bug fix, and one deployment at a time.
